# AntiFraudapp

Plataforma inteligente de prevenção de burlas baseada em Inteligência Artificial.

## 🚀 Sobre o projeto
O AntiFraudapp foi desenvolvido para detetar, prevenir e analisar tentativas de fraude em tempo real, protegendo utilizadores contra esquemas digitais, phishing e engenharia social.

## 🧠 Funcionalidades principais
- Deteção de mensagens fraudulentas com IA
- Análise de links suspeitos
- Sistema de alerta em tempo real
- Localização e identificação de atividades suspeitas
- Integração com tecnologia blockchain (ICP)
