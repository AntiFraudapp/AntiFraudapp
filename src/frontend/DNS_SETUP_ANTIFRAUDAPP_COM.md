# DNS Setup Instructions for antifraudapp.com

## ⚠️ Important Note

These DNS records must be configured by the domain owner in their domain registrar's control panel (e.g., GoDaddy, Namecheap, Cloudflare, etc.). This configuration cannot be done by the application developers.
